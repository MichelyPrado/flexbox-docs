Guia CSS: Flexbox

Flexbox é a abreviação de Flexible Box Module. Um modelo de layout que permite fácil controle da distribuição do espaço e alinhamento entre elementos HTML.

Neste guia, vamos exemplificar as principais propriedades possíveis para o elemento pai (o flex-container, no nosso caso, a "caixa" preta) e os elementos filhos (os flex-itens, no nosso caso, as "caixinhas" coloridas).

    Display:flex Flex-direction Justify-content Align-items Flex-wrap Flex-flow 

Display:flex

Vamos começar pela primeira propriedade que precisa ser declarada para que o Flexbox funcione, o Display:flex!

CSS 
.container{
   display:flex;
}

A propriedade display-flex vai organizar os flex-itens um ao lado do outro (equivalente ao valor "row") por padrão.
