Flexbox

Flexbox é a abreviação de Flexible Box Module. Um modelo de layout que permite fácil controle da distribuição do espaço e alinhamento entre elementos html.

display:flex

A primeira propriedade que precisa ser declarada para que o Flexbox funcione!

Algumas propriedades como flex-direction, justify-content e align-content já vem com os valores padrão.

Propriedades Valor
flex-direction row
justify-content flex-start
align-content flex-start

Exemplo de display:flex: faz com que o contêiner expanda toda a largura disponível
